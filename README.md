# Ovnivers — All-in-One Nuvio Addon

## Instalacion

1. Abri **Nuvio** > **Settings** > **Plugins**
2. Agrega esta URL:
   ```
   https://raw.githubusercontent.com/leonidas10009/ovnivers/main/manifest.json
   ```
3. Refresca y habilita los providers que quieras

## Desarrollo local

```bash
npm install
npm start        # Servidor en http://localhost:3000
```

Luego en Nuvio (modo debug) agregas `http://TU_IP:3000/manifest.json` en Plugin Tester.
